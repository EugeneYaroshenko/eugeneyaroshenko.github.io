Y-Productive: Desktop Application

Y-Productive (YP) is a SaaS startup that aims to help freelancers solve “How can I be more productive at my digital work?” problem. YP solves the problem by helping to leverage productive digital behaviors and providing guidance on how to eliminate unproductive ones with real-time feedback about a person’s work.
Being a React Developer in the Y-Productive team, I have built the application’s UI for macOS with other developers. I was also responsible for implementing user interactions inside the app.
